## Phase 0 ‚Äì Foundations (`00-foundations/`)

**Goals**
- Understand cloud security fundamentals
- Review networking concepts (TCP/IP, DNS, HTTP, firewalls)
- Learn IAM, authentication vs authorization
- Understand shared responsibility model

**Deliverables**
- Notes on cloud security & IAM
- Basic GCP security concepts
- Mini-lab: Secure a simple cloud resource with IAM

---

## Phase 1 ‚Äì SIEM & Logging (`01-siem/`)

- Understand SIEM architecture and data pipelines
- Learn log ingestion and normalization

- Understand alerting concepts

**Deliverables**
- SIEM concepts notes
- Log ingestion lab

## Phase 2 ‚Äì Detection Engineering (`02-detection-engineering/`)

**Deliverables**
- Detection engineering notes
- Sigma detection rules
- Detection tuning examples

## Phase 3 ‚Äì Incident Response (`03-incident-response/`)

**Deliverables**
- Incident response process notes

- Incident simulation lab
- Post-incident report template
